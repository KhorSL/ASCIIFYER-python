# Convert images to ASCII art with ASCIIFYER bot

## About ASCIIFYER
ASCIIFYER is a Telegram bot that converts images to ASCII art using [ASCII-ART](https://github.com/KhorSL/ASCII-ART)


Find the bot at [@S_ASCIIFYER_bot](https://t.me/S_ASCIIFYER_bot)
